      Local Image Generator Setup

      Checking for running image servers...

      Stability Matrix is a one-click manager for local AI image generation on Windows. It installs Python, CUDA drivers, and models automatically — no command line knowledge required.

      1. Click Download Stability Matrix below — this opens lykos.ai (official site)
      2. Download and run StabilityMatrix-win-x64.exe
      3. In Stability Matrix, click Add Package and choose ComfyUI or AUTOMATIC1111 Web UI
      4. Let it install — allow 15–30 minutes and about 20 GB of free disk space for models
      5. In the Models tab, download an SDXL model (e.g. Stable Diffusion XL Base 1.0)
      6. Click Launch in Stability Matrix to start your image server
      7. Click Test Connection below — this app will detect your server automatically
      Download Stability Matrix (lykos.ai)

      Install AUTOMATIC1111 Stable Diffusion Web UI manually. Requires Python 3.10 and Git. Recommended for users comfortable with the command line.

      1. Install Python 3.10 from python.org — tick "Add Python to PATH" during setup
      2. Install Git from git-scm.com
      3. Open a terminal and run:
        git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui
      4. Edit webui-user.bat — set COMMANDLINE_ARGS=--api (required for this app to connect)
      5. Run webui.bat — first run downloads ~5 GB of dependencies automatically
      6. Once running, the server is at http://127.0.0.1:7860
      7. Click Test Connection below

      Install ComfyUI — a node-based Stable Diffusion interface. More efficient than AUTOMATIC1111, but requires manual model placement and checkpoint name configuration.

      1. Download the ComfyUI standalone Windows package from github.com/comfyanonymous/ComfyUI/releases
      2. Extract the zip to a folder such as C:\ComfyUI
      3. Download an SDXL model (e.g. from civitai.com or Hugging Face) and place it in ComfyUI\models\checkpoints\
      4. Run run_nvidia_gpu.bat (or run_cpu.bat if you have no NVIDIA GPU)
      5. The server starts at http://127.0.0.1:8188
      6. Enter the exact checkpoint filename in the Model Name field below (e.g. sd_xl_base_1.0.safetensors)
      7. Click Test Connection below
